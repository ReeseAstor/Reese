# Reese Astor - Luxury Author Website

A sophisticated, elegant author website for Reese Astor, designed with a focus on minimalism, luxury aesthetics, and exceptional user experience.

## Features

- **Elegant Design**: Minimalist aesthetic with premium typography and refined color palette
- **Responsive Layout**: Seamlessly adapts to all device sizes
- **Smooth Animations**: Subtle scroll effects and hover interactions
- **Newsletter Integration**: Email subscription functionality
- **Contact Form**: Professional inquiry handling for readers, media, and events
- **Book Showcase**: Beautiful display of literary works with detailed information

## Pages

- **Home** (`index.html`) - Hero section, featured book, testimonials, and newsletter signup
- **About** (`about.html`) - Author biography, awards, and recognition
- **Books** (`books.html`) - Complete book collection with descriptions
- **Contact** (`contact.html`) - Contact form and book club resources

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (Cormorant Garamond, Montserrat)

## Getting Started

Simply open `index.html` in a web browser to view the website. No build process or dependencies required.

```bash
# Using Python's built-in server
python -m http.server 8000

# Using Node.js
npx serve
```
